Suggester læsekompas documentation

This service provides suggestions for content in læsekompasset/MinNæsteBog, through the following API:

Suggester part

/suggest{/e_book|audio_book}?query={q}

{/e_book|audio_book} is optional, and specifies whether you filter suggestions to either EBooks or Audio Books. If not specified, all types of books are suggested.

The query which suggestions are based on, is specified with ?query parameter.

Searcher part

/search?query={q}&field={author|title|workid|pid optional}&exact={optional default: false}&rows={optional, default: 10}

The query which suggestions are based on, is specified with ?query parameter.

field specifies which field should be searched, options are author, title, workid and pid. If no field is specified, a "full-text search" is assumed.

exact specifies whether the query should match exactly. This parameter is only in effect when field is specified. This is useful for retrieving a specific title, or documents from a specific author.

rows specifies the maximum number of rows returned, defaults to 10.
